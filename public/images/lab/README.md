# Lab image folders

Place evidence images into the following folders:

- `lab-001/` for AWS Free Tier screenshots
- `lab-002/` for AWS IAM screenshots
- `lab-003/` for Amazon VPC screenshots

Recommended naming:

- `lab-001-01.png`, `lab-001-02.png`, ...
- `lab-002-01.png`, `lab-002-02.png`, ...
- `lab-003-01.png`, `lab-003-02.png`, ...
